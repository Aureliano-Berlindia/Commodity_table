# 🌈爬虫商品展示初版
   *Commodity Table Demo，still developing*
   
---

## 💎目前功能:

* 通过`Brand`筛选`Country`选项列表，反之亦然
* 通过`Brand` & `Country` 筛选`Model`列表
* **Model callback**中使用`Brand`,`Country`,`Model`确定唯一商品，展示在第一列
* 设置初始显示商品
* 在选中指定商品后再选择All，目前显示商品不变
